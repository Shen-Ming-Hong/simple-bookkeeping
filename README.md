# 記帳與財務預測

這是一個純前端的個人記帳與財務預測工具，用來整理多帳戶固定收支、分期付款、固定每月轉帳、指定月份轉帳與未來月度資金變化。

正式使用網址：`https://money.esperar.org/`

## 功能

- 管理多個銀行帳戶，每個帳戶都有獨立名稱與期初餘額
- 管理定期收入，支援每月與每年固定月份，且必須歸戶
- 管理支出規劃，支援每月、每年固定月份與指定年月一次，且必須歸戶
- 管理分期付款，依剩餘月數自動計入未來支出，且必須歸戶
- 管理固定每月轉帳，模擬把錢從某帳戶固定存到另一個帳戶
- 管理指定月份轉帳，處理單次且月份不固定的帳戶間資金調度
- 支援收入、支出、分期的批次勾選歸戶
- 顯示總表與帳戶月度明細表，並提供總餘額趨勢圖
- 匯出與匯入 JSON 備份，支援舊版資料自動轉換
- 可選擇連接 Google Drive 做背景同步，保留本機快取作為快速啟動來源
- 支援以 PWA 安裝到桌面或手機主畫面，完成首次載入後可離線啟動並查看本機資料與圖表
- 支援亮色與暗色主題切換，主題偏好會保存在目前瀏覽器

## 使用方式

1. 開啟網站後先設定預測月數，並建立自己的帳戶與期初餘額。
2. 依照情況新增收入、支出、分期，指定各自使用的帳戶。
3. 如果有固定每月或單次指定月份要轉到別的帳戶，再新增對應的轉帳規劃。
4. 需要調整大量資料時，可在收入、支出、分期列表勾選多筆後批次歸戶。
5. 查看總表、帳戶月度明細與圖表，確認每月淨額與各帳戶月底餘額。
6. 需要保留或搬移資料時，使用匯出與匯入 JSON 備份功能。

## 資料過渡

- 舊版單一存款資料會在載入時自動升版為多帳戶格式
- 舊資料的存款會搬進「主帳戶」
- 舊版收入、支出、分期會自動掛到「主帳戶」
- 舊版多帳戶資料會保留原本的固定每月轉帳，並自動補上指定月份轉帳欄位
- 匯入舊版 JSON 備份時也會自動轉換，不需要先手動整理

## 資料儲存方式

- 預設資料會先保存在當前瀏覽器的本機儲存
- 亮色 / 暗色主題偏好也會保存在同一個瀏覽器的本機儲存
- 若有設定並授權 Google Drive，頁面會在背景把資料同步到你自己的 Google Drive
- 啟動時會先載入本機快取，再嘗試背景比對雲端資料
- 雲端資料時間比本機新時，會自動把雲端資料同步回本機
- 本機資料時間比雲端新時，不會在啟動當下直接覆蓋雲端；後續變更或手動上傳才會推送
- 不會上傳到自家伺服器，Drive 同步直接由瀏覽器呼叫 Google API
- 清除瀏覽器資料後，本機快取會消失；若先前已有同步到 Drive，重新連線後仍可拉回資料
- 若要跨裝置搬移資料，也可繼續使用 JSON 備份匯出與匯入

## Google Drive 同步

1. 點擊右上角齒輪按鈕，開啟同步面板。
2. 點擊「連線 Google Drive」，依畫面指示完成 Google 帳號授權。
3. 授權後，資料會在背景自動同步到你自己的 Google Drive，無需額外操作。

### 同步說明

- 同步檔案會儲存在 Google Drive 的隱藏應用程式資料夾，不會出現在一般 Drive 檔案列表
- 為了提高重新整理頁面與重開瀏覽器後的自動恢復率，短期 Google 授權會暫存在目前裝置的瀏覽器本機儲存
- 首次手動授權時，會額外記住這個應用上次使用的 Google 帳號資訊，之後手動重新連接時會優先沿用同一帳號
- 應用不會讀取 Chrome 或瀏覽器目前登入的 Google 帳號，只會使用自己上次記住的授權帳號提示
- 啟動時會優先嘗試恢復先前的 Google Drive 連線；若權杖仍有效，會直接接續同步
- 若權杖已過期，頁面不會再於背景自動打開 Google 重新授權；會直接改成等待你手動重新連線，避免手機上出現自動跳出後又消失的登入視窗
- 這種做法能提高便利性，但不保證永久免重新授權；若 Google 授權失效、權杖過期或瀏覽器政策限制靜默授權，仍需要手動重新連接

## 注意事項

- 網站可公開訪問，但沒有登入保護，請不要把網址當成私密保護機制
- 這個工具適合做個人收支與現金流預估，不等同正式會計或銀行對帳資料
- 圖表腳本會跟著站點一起快取，首次載入成功後，離線仍可顯示圖表
- PWA 的完整離線能力需先在線上成功開啟過一次，之後才會把必要靜態資源快取到目前裝置
