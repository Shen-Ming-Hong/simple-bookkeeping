# 記帳與財務預測

這是一個純前端的個人記帳與財務預測工具，用來整理固定收支、分期付款與未來月度資金變化。

正式使用網址：`https://money.esperar.org/`

## 功能

- 設定目前存款與預測月數
- 管理定期收入，支援每月與每年固定月份
- 管理定期支出，支援每月與每年固定月份
- 管理分期付款，依剩餘月數自動計入未來支出
- 顯示未來月度預測表格與餘額趨勢圖
- 匯出與匯入 JSON 備份

## 使用方式

1. 開啟網站並輸入目前存款與預測月數。
2. 依照自己的情況新增定期收入與定期支出。
3. 若有分期付款，再補上每月金額與剩餘月數。
4. 查看預測結果表格與圖表，確認每月淨額與月末餘額變化。
5. 需要保留或搬移資料時，使用匯出與匯入 JSON 備份功能。

## 資料儲存方式

- 資料只存在當前瀏覽器的 `localStorage`
- 不會上傳到伺服器
- 不同裝置或不同瀏覽器之間不會自動同步
- 清除瀏覽器資料後，本機內容會消失
- 若要跨裝置搬移資料，請使用 JSON 備份匯出與匯入

## 注意事項

- 網站可公開訪問，但沒有登入保護，請不要把網址當成私密保護機制
- 這個工具適合做個人收支與現金流預估，不等同正式會計或銀行對帳資料
- 圖表依賴 CDN 載入 Chart.js，若網路阻擋 CDN，圖表可能無法顯示
